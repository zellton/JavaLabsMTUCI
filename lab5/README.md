Лабораторная работа №5: Регулярные выражения в Java

Описание

Данная лабораторная работа посвящена изучению регулярных выражений в Java.  
В ходе выполнения работы были реализованы программы для поиска и проверки данных с использованием классов Pattern и Matcher.

Программа демонстрирует работу с шаблонами поиска, проверкой корректности данных, заменой фрагментов текста и обработкой ошибок регулярных выражений.

Ключевые концепции

Regular Expressions — механизм поиска и обработки текста по шаблону

&nbsp;&nbsp;&nbsp;&nbsp;Pattern — класс для компиляции регулярного выражения

&nbsp;&nbsp;&nbsp;&nbsp;Matcher — класс для поиска совпадений в тексте

&nbsp;&nbsp;&nbsp;&nbsp;PatternSyntaxException — обработка ошибок регулярных выражений

&nbsp;&nbsp;&nbsp;&nbsp;replaceAll() — замена найденных совпадений

&nbsp;&nbsp;&nbsp;&nbsp;matches() и find() — методы поиска совпадений

Задания:

1 Поиск всех чисел в тексте

&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;Создана программа для поиска целых и дробных чисел в строке.

&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;Использовано регулярное выражение для поиска чисел.

&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;Реализована обработка ошибок и проверка пустого текста.

2 Проверка корректности пароля

&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;Создана программа для проверки сложности пароля.

&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;Пароль должен содержать заглавную букву, цифру и иметь длину от 8 до 16 символов.

&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;Использованы lookahead-конструкции регулярных выражений.

3 Поиск строчной буквы перед заглавной

&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;Реализован поиск случаев, когда после строчной буквы идет заглавная.

&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;Найденные сочетания выделяются символами "!".

&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;Использован метод replaceAll().

4 Проверка корректности IP-адреса

&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;Создана программа для проверки IPv4-адресов.

&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;Проверяется диапазон чисел от 0 до 255.

&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;Использованы сложные шаблоны регулярных выражений.

5 Поиск слов на заданную букву

&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;Реализован поиск слов, начинающихся с указанной буквы.

&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;Поддерживается поиск независимо от регистра символов.

&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;Используются границы слова и динамическое построение шаблона.

6 Демонстрация работы программы

&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;Продемонстрирована работа методов find(), matches() и replaceAll().

&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;Показан поиск совпадений и проверка различных строк.

&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;Реализована обработка ошибок регулярных выражений.
