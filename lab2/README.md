Лабораторная работа №2: Объектно-ориентированное программирование в Java

Описание

Данная лабораторная работа посвящена изучению основных принципов объектно-ориентированного программирования в Java.  
В ходе выполнения работы была реализована иерархия классов сотрудников компании с использованием наследования, абстракции, инкапсуляции, перегрузки и переопределения методов.

Программа демонстрирует создание объектов различных классов, работу конструкторов, геттеров и сеттеров, а также использование статических полей и методов.

Ключевые концепции

Абстракция — создание базового абстрактного класса Employee

&nbsp;&nbsp;&nbsp;&nbsp;Наследование — расширение классов Employee → Manager → Administrator

&nbsp;&nbsp;&nbsp;&nbsp;Инкапсуляция — использование private полей и методов доступа

&nbsp;&nbsp;&nbsp;&nbsp;Полиморфизм — переопределение методов showInfo() и DoItsJob()

&nbsp;&nbsp;&nbsp;&nbsp;Перегрузка методов — реализация метода fixBugs() с разными параметрами

&nbsp;&nbsp;&nbsp;&nbsp;Static — использование статического счетчика объектов

Задания:

1 Создание абстрактного класса Employee

&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;Создан базовый абстрактный класс Employee.

&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;Класс содержит поля: имя сотрудника, зарплата и идентификационный номер.

&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;Реализованы конструкторы, геттеры и сеттеры.

&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;Добавлен статический счетчик созданных объектов.

2 Реализация класса Programmer

&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;Создан класс Programmer, наследующий Employee.

&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;Добавлены поля: основной язык программирования, категория и участие в проекте.

&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;Реализовано переопределение методов showInfo() и DoItsJob().

&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;Продемонстрирована перегрузка метода fixBugs().

3 Реализация класса Manager

&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;Создан класс Manager, наследующий Employee.

&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;Добавлены поля: размер команды, премия и название отдела.

&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;Реализован метод calculateAnnualSalary() для расчета годового дохода.

4 Реализация класса Administrator

&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;Создан класс Administrator, наследующий Manager.

&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;Добавлены поля: название сервера, уровень доступа и статус дежурства.

&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;Реализован метод rebootServer() для перезагрузки сервера.

5 Демонстрация работы программы

&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;Созданы объекты Programmer, Manager и Administrator.

&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;Продемонстрирована работа методов классов и вывод информации об объектах.

&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;Проверена работа статического счетчика сотрудников.
