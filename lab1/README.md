Лабораторная работа №1: Введение в Java

Описание

Данная лабораторная работа посвящена изучению базовых возможностей языка Java.  
В ходе выполнения работы были реализованы простые консольные программы, демонстрирующие работу с методами, циклами, строками и условиями.

Ключевые концепции

System.out.println() — вывод информации в консоль

&nbsp;&nbsp;&nbsp;&nbsp;Метод main() — точка входа в программу

&nbsp;&nbsp;&nbsp;&nbsp;Циклы for — перебор данных и повторение операций

&nbsp;&nbsp;&nbsp;&nbsp;Методы — разделение логики программы на отдельные функции

&nbsp;&nbsp;&nbsp;&nbsp;String — работа со строками

&nbsp;&nbsp;&nbsp;&nbsp;Условные операторы — проверка условий выполнения

Задания:

1 Программа Hello World

&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;Создана первая Java-программа, выводящая строку "Hello World" в консоль.

&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;Изучена структура Java-программы и работа метода main().

2 Проверка строки на палиндром

&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;Реализована программа для проверки строк на палиндром.

&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;Создан метод reverseString() для переворота строки.

&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;Создан метод isPalindrome() для сравнения строки с её перевернутой версией.

&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;Аргументы передаются через командную строку.

3 Поиск простых чисел

&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;Реализована программа для поиска всех простых чисел меньше 100.

&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;Создан метод isPrime() для проверки числа на простоту.

&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;Для проверки используются циклы и операция остатка от деления.
