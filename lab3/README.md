Лабораторная работа №3: Хэш-таблицы в Java

Описание

Данная лабораторная работа посвящена изучению структуры данных HashMap и принципов работы хэш-таблиц в Java.  
В ходе выполнения работы была реализована собственная хэш-таблица с методом цепочек, а также система хранения заказов интернет-магазина с использованием встроенного класса HashMap.

Программа демонстрирует операции добавления, поиска и удаления элементов, а также работу с коллекциями и обобщениями (Generics).

Ключевые концепции

HashTable — структура данных для хранения пар «ключ-значение»

&nbsp;&nbsp;&nbsp;&nbsp;HashMap — встроенная реализация хэш-таблицы в Java

&nbsp;&nbsp;&nbsp;&nbsp;Коллизии — ситуация совпадения хэшей разных ключей

&nbsp;&nbsp;&nbsp;&nbsp;Метод цепочек — способ хранения элементов через связанные списки

&nbsp;&nbsp;&nbsp;&nbsp;Generics — обобщенные типы данных

&nbsp;&nbsp;&nbsp;&nbsp;Collection Framework — работа с коллекциями Java

Задания:

1 Реализация собственной хэш-таблицы

&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;Создан обобщенный класс HashTable<K, V>.

&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;Реализовано хранение данных с помощью массива связных списков.

&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;Добавлены методы put(), get() и remove() для работы с элементами.

&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;Реализованы методы size() и isEmpty().

&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;Использован механизм разрешения коллизий методом цепочек.

2 Работа со встроенным классом HashMap

&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;Создана система хранения заказов интернет-магазина.

&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;Реализован класс Product для хранения информации о товаре.

&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;Создан класс Order для хранения информации о заказе и списке товаров.

&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;Создан класс OrderStorage для управления заказами через HashMap.

3 Реализация операций с заказами

&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;Реализовано добавление новых заказов.

&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;Добавлена проверка на дублирование заказов.

&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;Реализован поиск заказа по номеру.

&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;Реализовано удаление заказов из хранилища.

&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;Добавлен вывод всех сохраненных заказов.

4 Демонстрация работы программы

&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;Созданы несколько заказов с товарами.

&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;Продемонстрированы операции вставки, поиска и удаления элементов.

&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;Показана работа собственной хэш-таблицы и встроенного HashMap.
