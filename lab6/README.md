Лабораторная работа №6: Коллекции и дженерики в Java

Описание

Данная лабораторная работа посвящена изучению коллекций, дженериков и работы со структурами данных в Java.  
В ходе выполнения работы были реализованы программы для подсчета частоты слов в тексте, создания собственного обобщенного стека и системы учета продаж товаров.

Программа демонстрирует использование коллекций Map, List, LinkedHashMap, а также применение Generics и сортировки данных.

Ключевые концепции

Collection Framework — набор коллекций для хранения и обработки данных

&nbsp;&nbsp;&nbsp;&nbsp;Map — структура данных «ключ-значение»

&nbsp;&nbsp;&nbsp;&nbsp;LinkedHashMap — коллекция с сохранением порядка добавления элементов

&nbsp;&nbsp;&nbsp;&nbsp;Generics — обобщенные типы данных

&nbsp;&nbsp;&nbsp;&nbsp;List — упорядоченная коллекция элементов

&nbsp;&nbsp;&nbsp;&nbsp;Сортировка коллекций — обработка и упорядочивание данных

Задания:

1 Подсчет самых частых слов в тексте

&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;Создана программа TopWords для анализа текстового файла.

&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;Использована коллекция HashMap для хранения слов и количества их повторений.

&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;Реализована очистка слов от знаков пунктуации.

&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;Добавлена сортировка слов по убыванию частоты встречаемости.

&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;Выводится топ-10 наиболее популярных слов.

2 Реализация обобщенного класса Stack<T>

&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;Создан собственный класс Stack<T> с использованием Generics.

&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;Реализованы методы push(), pop() и peek().

&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;Добавлена проверка переполнения и пустоты стека.

&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;Использован массив для хранения элементов стека.

3 Реализация системы учета продаж

&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;Создан класс SalesTracker для учета проданных товаров.

&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;Использована коллекция LinkedHashMap для хранения товаров и количества продаж.

&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;Реализован подсчет общей суммы продаж.

&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;Добавлен поиск наиболее популярного товара.

4 Работа с текстовыми данными

&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;Программа анализирует текстовый файл и подсчитывает количество повторений слов.

&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;Используется приведение слов к нижнему регистру.

&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;Удаляются символы пунктуации перед анализом текста.

5 Демонстрация работы программы

&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;Продемонстрирована работа коллекций Map, List и LinkedHashMap.

&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;Показана работа обобщенного стека.

&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;Реализован анализ текста и учет продаж товаров.
