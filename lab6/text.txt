Java — это объектно-ориентированный язык программирования. Java позволяет писать эффективный код. 
Программа на Java компилируется в байт-код. Объект в Java является основой всего. 
Каждый объект имеет состояние и поведение. Код должен быть чистым. 
Программа работает быстро, если код оптимизирован. Java широко используется в мире. 
Объект и класс — важные понятия. Напишите программу, которая анализирует текст. 
Текст содержит много слов. Программа считает слова. Java код, Java объект, Java программа.
Код, код и еще раз код. Объектно-ориентированный подход помогает структурировать код.
